Bienvenido al croquis de territorios

Creada por Bryan Levi

Esta pagina ayuda a revisar cada localidad, colorear las zonas del territorio, agregar nombres de colonias y generar el PDF final para imprimir o guardar.

1Elige la localidad y revisa el mapa base con calles, avenidas y datos visuales.
2Selecciona color, pinta las areas necesarias y guarda los cambios antes de pasar al siguiente territorio.
3Cuando el croquis este listo, usa PDF para preparar la hoja de impresion.

Guia rapida de herramientas

Restablecer recuadroRegresa el tamano del recuadro del territorio al valor inicial.
Color y transparenciaDefine el color de relleno y que tan visible queda sobre el mapa.
Editar pintada o textoSelecciona una zona o un texto que ya habias marcado para modificarlo.
Editar contornoAjusta los puntos del borde de una zona coloreada.
Pintar areaToca varios puntos del mapa para formar la zona que quieres colorear.
Agregar coloniaColoca texto en el mapa. Usa / para separar el nombre en dos renglones.
Guardar cambiosGuarda colores y textos del territorio actual.
DeshacerRegresa el ultimo punto o cambio guardado si necesitas corregir.
BorrarElimina colores o textos del territorio que estas editando.
PDFPrepara la hoja del croquis para imprimir o guardar.
Menu / instruccionesRegresa a esta pantalla para consultar la guia cuando lo necesites.
Agregar territorioMarca dos esquinas para formar el recuadro y despues escribe el nombre.
Eliminar territorioQuita el territorio actual de la lista despues de confirmar.
